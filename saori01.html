Ciberseguridad 01

Tipos de Malware

Tipo de Malware Función Impacto
Virus Se autorreplica e infecta archivos Daño de datos y del sistema
Gusanos Se propagan a través de las redes Interrupción de red, agotamiento de recursos
Caballo de Troya Se disfraza de software legítimo Robo de datos, control del sistema
Spyware Supervisa en secreto la actividad del usuario Violación de privacidad, robo de información
Adware Muestra anuncios no deseados Molestias y bajo rendimiento
Rootkit Oculta software malicioso Acceso no autorizado, amenaza persistente
Keylogger Registra pulsaciones del teclado Robo de credenciales, filtración de datos
Fileless Malware Opera directamente en la memoria Ataques sigilosos y difíciles de detectar
Cryptojacker Minado no autorizado de criptomonedas Consumo de recursos, lentitud del sistema
Botnet Crea redes de dispositivos infectados Ataques DDoS, distribución de spam
Bomba Lógica Ejecuta código al cumplirse una condición Destrucción de datos, daños al sistema
Wiper Elimina datos de forma irreversible Pérdida total de información
